# 🎬 BingeWatch – Movie Recommendation System

A Content-Based Movie Recommender System that suggests movies similar to user preferences and performs sentiment analysis on user reviews.

---

## 🚀 Features
- Recommend similar movies using content-based filtering
- Sentiment analysis on IMDb user reviews
- Real-time movie search with AJAX
- Fetch movie details (title, genre, rating, poster) using TMDB API
- Interactive and user-friendly interface

---

## 🧠 How It Works
- Uses **TF-IDF Vectorization** to convert text into numerical form
- Computes similarity using **Cosine Similarity**
- Scrapes IMDb reviews using **BeautifulSoup**
- Performs **Sentiment Analysis** on reviews
- Displays recommended movies based on similarity score

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy, Scikit-learn
- BeautifulSoup (Web Scraping)
- TMDB API
- HTML, CSS, JavaScript
- AJAX
